Javascript Language

Explain to a non-tech friend

  • In the Javascript language, describe all the use cases for each of:
    • parenthesis () - eg '1. Execute or start a function var result = doSomething()'

      • These are used for starting a function and can be used to pass values or strings called parameters to the function for example function doSomething(a, b). They can also be used to control the order of arithmetic. For example grouping expressions a = (4 + 3) * 2 or containing expressions (a < 10)

    • brackets []

      • In Javascript brackets are used for either accessing data from an array or properties of an object. The process behind this is called bracket notation. As an example, we can use var data = array[0] to get access to the first data in an array. You can use objectname["property"], to access an object that stores data through properties. The object name and property can also be used in a looping condition for example board.cells[i].isMine, to access a number of mines until you can't.

    • braces {}

      • Braces are used to let you know where the start and end of the function is, for example, function doSomething() {code}. They are also used to seperate blocks of code within a function for example a for loop in function doSomething() { for () {} }. Another use of braces can be for objects for example object = {property: [value,..]}

    • single quotes '

      • Single quotes are mostly used with classes and events in terms of how they respond to user action. They are used for strings, which are pieces of data, in the form of a sequence of characters, that can be made up of words or numbers. Single quotes can also be used to put HTML within Javascript and they can be used within double quotes to highlight text within text.

    • double quotes ""

      • Double quotes are used for strings to display text. The main use of double quotes in javascript is when we want to create a variable and give it a value. Double quotes can be used in the same way as single quotes for displaying text, however the main difference is that single quotes can go either inside or outside double quotes. Double quotes cannot go inside double quotes, unless they have an escape character with a backslash \ . For example
      "I \"love\" Javascript"
